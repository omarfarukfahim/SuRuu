About Your SuRuu

Project Title

Your SuRuu: The Friendly AI Chat Companion

Subtitle

Chat, create, and learn with an AI designed to be your helpful friend.

Project Information

This project is a full-stack web application built with a Python Flask backend and a modern HTML, CSS, and JavaScript frontend. It integrates with the Google Gemini API to provide intelligent chat responses and (simulated) image generation.

Core Features:

  • Generative AI Chat: Have natural, free-flowing conversations.
  • Image Generation Mode: Use the /imagine command to get detailed descriptions and placeholder images.
  • SuRuu Magic: A special dashboard (in development) to house powerful tools:
    • AI Calculator: Solves math problems from natural language.
    • SuRuu Note: A simple place to jot down ideas.
    • PDF Assistant: Upload PDFs to ask questions and generate flashcards.
  • Responsive Design: Fully functional and beautiful on all devices, from phones to desktops.
